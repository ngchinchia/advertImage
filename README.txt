This git repository features advertisement image auto scrolling for both android and ios.

Features:
- Auto scroll to next image in 2.5s
- Manual scrollable in for both android and ios
- Opens specific url links to each images on press

Technologies: 
- React Native Expo
- Redux to store global state 

Instructions: 
- Simply clone git repository
- Run npm install
- Run npx expo start
- Scan the QR code / open in emulator by pressing respective command.
